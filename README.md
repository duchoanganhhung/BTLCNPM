## Setup

# Chạy server

  `cd be` <br> 
  `python -m venv .venv` <br>
  `.venv\Scripts\Activate.ps1` HOẶC `.venv\bin\activate`<br>
  `pip install -r requirements.txt`<br>
  `python app.py`

# Chạy app
  `cd FE` <br>

  Run `npm i` to install the dependencies. <br>

  Run `npm run dev` to start the development server.
